 ▪ 以下是 Hermes 网关中邮箱配置的完整实现分析：                                                                                                                                                                    
                                                                                                                                                                                                                   
   ---                                                                                                                                                                                                             
                                                                                                                                                                                                                   
   架构概览                                                                                                                                                                                                        
                                                                                                                                                                                                                   
   邮箱适配器位于 gateway/platforms/email.py，使用 IMAP 收信 + SMTP 发信 的双协议模型，继承自 BasePlatformAdapter。                                                                                                
                                                                                                                                                                                                                   
   ---                                                                                                                                                                                                             
                                                                                                                                                                                                                   
   配置方式                                                                                                                                                                                                        
                                                                                                                                                                                                                   
   所有邮箱配置通过 .env 环境变量 驱动（不是 config.yaml），在 gateway/config.py 中自动检测并启用：                                                                                                                
                                                                                                                                                                                                                   
   必需的环境变量                                                                                                                                                                                                 
   ┌─────────────────┬────────────────────┬─────────────────────┐                                                                                                                                                  
   │ 变量            │ 说明               │ 示例                │                                                                                                                                                  
   ├─────────────────┼────────────────────┼─────────────────────┤                                                                                                                                                  
   │ EMAIL_ADDRESS   │ 代理邮箱地址       │ hermes@gmail.com    │                                                                                                                                                  
   │ EMAIL_PASSWORD  │ 密码或应用专用密码 │ xxxx xxxx xxxx xxxx │                                                                                                                                                  
   │ EMAIL_IMAP_HOST │ IMAP 服务器        │ imap.gmail.com      │                                                                                                                                                  
   │ EMAIL_SMTP_HOST │ SMTP 服务器        │ smtp.gmail.com      │                                                                                                                                                  
   └─────────────────┴────────────────────┴─────────────────────┘                                                                                                                                                  
   可选环境变量                                                                                                                                                                                                    
   ┌─────────────────────────┬────────┬──────────────────────────────────────────┐                                                                                                                                 
   │ 变量                    │ 默认值 │ 说明                                     │                                                                                                                                 
   ├─────────────────────────┼────────┼──────────────────────────────────────────┤                                                                                                                                 
   │ EMAIL_IMAP_PORT         │ 993    │ IMAP 端口（SSL）                         │                                                                                                                                 
   │ EMAIL_SMTP_PORT         │ 587    │ SMTP 端口（STARTTLS）                    │                                                                                                                                 
   │ EMAIL_POLL_INTERVAL     │ 15     │ 轮询间隔（秒）                           │                                                                                                                                 
   │ EMAIL_ALLOWED_USERS     │ (空)   │ 逗号分隔的允许发件人地址白名单           │                                                                                                                                 
   │ EMAIL_ALLOW_ALL_USERS   │ false  │ 设为 true 则允许所有发件人               │                                                                                                                                 
   │ EMAIL_HOME_ADDRESS      │ (空)   │ "家庭频道"地址（用于 home channel 功能） │                                                                                                                                 
   │ EMAIL_HOME_ADDRESS_NAME │ "Home" │ 家庭频道显示名                           │                                                                                                                                 
   └─────────────────────────┴────────┴──────────────────────────────────────────┘                                                                                                                                 
   自动启用逻辑 (gateway/config.py:1468-1489)                                                                                                                                                                      
    email_addr = os.getenv("EMAIL_ADDRESS")                                                                                                                                                                        
    email_pwd = os.getenv("EMAIL_PASSWORD")                                                                                                                                                                        
    email_imap = os.getenv("EMAIL_IMAP_HOST")                                                                                                                                                                      
    email_smtp = os.getenv("EMAIL_SMTP_HOST")                                                                                                                                                                      
    if all([email_addr, email_pwd, email_imap, email_smtp]):                                                                                                                                                       
        # 自动创建 PlatformConfig 并 enabled = True                                                                                                                                                                
        config.platforms[Platform.EMAIL].enabled = True                                                                                                                                                            
   只要四个必需变量全部设置，邮箱平台就会自动启用——无需手动在 config.yaml 中开启。                                                                                                                                 
                                                                                                                                                                                                                   
   config.yaml 可选配置                                                                                                                                                                                            
    platforms:                                                                                                                                                                                                     
      email:                                                                                                                                                                                                       
        skip_attachments: true   # 忽略所有附件（安全/带宽考虑）                                                                                                                                                   
   ---                                                                                                                                                                                                             
                                                                                                                                                                                                                   
   消息流转                                                                                                                                                                                                        
                                                                                                                                                                                                                   
   收信（IMAP → Agent）                                                                                                                                                                                            
                                                                                                                                                                                                                   
   1. connect() — 启动时测试 IMAP/SMTP 连接，将已有邮件标记为已读（_seen_uids），然后启动 _poll_loop 异步任务                                                                                                      
   2. _poll_loop() — 每 EMAIL_POLL_INTERVAL 秒调用 _check_inbox()                                                                                                                                                  
   3. _fetch_new_messages() — 在线程池中运行 IMAP UNSEEN 搜索，提取正文、附件、发件人信息                                                                                                                         
   4. _dispatch_message() — 过滤自检：                                                                                                                                                                             
     - 跳过自发邮件（发件人 = 自己）                                                                                                                                                                               
     - 跳过自动发件人（noreply、mailer-daemon 等模式 + Auto-Submitted 等 RFC 头）                                                                                                                                  
     - 跳过白名单外的发件人（如 EMAIL_ALLOWED_USERS 已设置）                                                                                                                                                       
     - 构建 MessageEvent 交给 handle_message() → 进入标准网关消息管道                                                                                                                                              
                                                                                                                                                                                                                   
   发信（Agent → SMTP）                                                                                                                                                                                            
                                                                                                                                                                                                                   
   - send() — 纯文本回复，自动添加 Re: 前缀和 In-Reply-To/References 头实现邮件线程                                                                                                                                
   - send_document() — 带附件发送                                                                                                                                                                                  
   - send_multiple_images() — 多图附件批量发送                                                                                                                                                                     
   - send_image() — 图片 URL 写入正文（不下载远程图片）                                                                                                                                                            
                                                                                                                                                                                                                   
   所有 SMTP 发送均使用 STARTTLS (smtp.starttls())。                                                                                                                                                               
                                                                                                                                                                                                                   
   特殊处理                                                                                                                                                                                                        
                                                                                                                                                                                                                   
   - 163/NetEase 邮箱兼容：_send_imap_id() 发送 RFC 2971 IMAP ID 命令，否则该邮箱会返回 BYE Unsafe Login                                                                                                           
   - UID 内存管理：_seen_uids 集合上限 2000 条，超出后保留最新的一半                                                                                                                                               
   - 线程上下文：_thread_context 字典记住每个对话的最后 subject + message-id，确保回复在同一邮件线程中                                                                                                             
                                                                                                                                                                                                                   
   ---                                                                                                                                                                                                             
                                                                                                                                                                                                                   
   访问控制                                                                                                                                                                                                        
                                                                                                                                                                                                                   
   网关层统一管理（gateway/run.py）：                                                                                                                                                                              
                                                                                                                                                                                                                   
   - EMAIL_ALLOWED_USERS — 逗号分隔白名单                                                                                                                                                                          
   - EMAIL_ALLOW_ALL_USERS — 设为 true 跳过白名单检查                                                                                                                                                              
                                                                                                                                                                                                                  
   适配器的 _dispatch_message() 中也有一层提前过滤，防止在白名单拒绝的情况下仍然发出回复（竞态条件防护）。
